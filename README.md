wdev-vagrant-zf2
================

Vagrant, ZF2, Apigility and Angular.


 ## Instalation

 1) Git clone git clone https://github.com/fwahlqvist/wdev-vagrant-zf2
 2) cd wdev-vagrant-zf2
 3) vagrant ssh
 4) cd /var/www 
 5) sudo ./bin/init-build
